# Regressão Linear: Marketing vs. Vendas

Este repositório guarda a análise que fiz para verificar como o investimento em marketing se relaciona com o número de vendas.  
O objetivo foi entender se existe uma relação linear, quanto cada mil reais investidos aumentam nas vendas e fazer algumas previsões.

## O que foi feito

- Criação do conjunto de dados
- Ajuste do modelo de regressão linear
- Cálculo dos coeficientes
- Gráficos e análise dos resíduos
- Previsões para valores específicos
- Estimativa de investimento para alcançar 100 vendas

Tudo está explicado no arquivo:

**Relatorio_Analise_Marketing.md**

## Principais resultados

- Cada R$ 1.000 investidos geram em média cerca de 3,18 vendas.
- Para R$ 20.000 investidos, a previsão fica em torno de 86 vendas.
- O modelo ficou com R² de 0.984.
- O investimento aproximado para atingir 100 vendas é de R$ 24,27 mil.

## Tecnologias

Python, Pandas, NumPy, Scikit-learn e Matplotlib.
